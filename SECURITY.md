# Security Policy

## Supported Versions

Currently, only the latest stable release of CYMOE is actively supported and receives security updates.

Older versions may not receive security patches. Users are strongly encouraged to upgrade to the most recent stable version.

## Reporting a Vulnerability

If you discover a security vulnerability in CYMOE, please do not disclose it publicly before a fix is released.

Instead, report the issue directly via:
